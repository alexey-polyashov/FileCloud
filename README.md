# FileCloud

Разработка простого сетевого хранилища файлов. Проект реализован на JAVA 8.

Bспользуемые библиотеки и технологии:
 - JavaFX
 - база данных MySQL на sqlite.JDBC для хранения списка пользователей
 - библиотека JavaMail API
 - Netty
 - Lombok
